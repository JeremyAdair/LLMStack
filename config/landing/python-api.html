Python API

FastAPI service for toolbox jobs and health checks.

Back to Docs Back to Landing

Endpoints

  • Docs: http://localhost:8000/docs
  • OpenAPI JSON: http://localhost:8000/openapi.json
  • Health: http://localhost:8000/health
  • Scripts UI: http://localhost:8000/ui
  • RAG ingest (one-shot): POST http://localhost:8000/rag/ingest_once
  • PLC poll (one-shot): POST http://localhost:8000/plc/poll_once

Example call:

curl -X POST http://localhost:8000/rag/ingest_once
