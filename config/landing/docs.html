Python API

Core services

  • Redis: available to containers at redis:6379
  • Prometheus: http://prometheus:9090 (internal)
  • Ollama: http://ollama:11434 (internal)
  • Qdrant: http://qdrant:6333 (internal)

Local admin access is available on the host ports shown in the stack status. These services are meant to be used from other containers unless you explicitly publish ports.
